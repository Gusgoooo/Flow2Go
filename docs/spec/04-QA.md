## Flow2Go 验收用例（QA Checklist）

> 建议每次改动 Frame/Group/Edge 相关逻辑后跑一遍。

### A. 默认示例与初始化

- [ ] 新用户空工程进入：自动加载默认示例（不是空白画布）
- [ ] 默认示例 viewport 正常（不强制回到 (0,0,1)）

### B. Frame（画框）容器语义

#### B1. 拖入（Drag-in，以光标为准）
- [ ] 拖动普通节点到 Frame 内，松手时光标在 Frame 内 → 节点成为 Frame 子集（parentId 正确）
- [ ] 拖动群组/Frame 到另一个 Frame 内，松手光标在目标 Frame 内 → 成为其子集
- [ ] 多选拖动（包含多个节点/群组/素材）：只重挂载顶层，子树不被打散

#### B2. 拖出（Drag-out，以光标为准）
- [ ] 从 Frame 内拖出，松手光标不在父 Frame 内 → 脱离父
- [ ] A 包 B 包 C：从 C 拖出但光标落在 B 内 → 重挂载到 B
- [ ] A 包 B 包 C：从 C 拖出但光标落在 A 内 → 重挂载到 A
- [ ] 从任意层拖出，光标落在任何 Frame 外 → 脱离到根层

#### B3. 防循环
- [ ] 拖拽不会产生 parentId 循环（出现循环时应拒绝挂载）

#### B4. 删除画框保留内容
- [ ] 点击画框菜单：删除画框但保留内部元素
- [ ] 删除后元素不跳位（绝对位置不变）
- [ ] 删除后元素的 parentId 变为画框的 parentId（或根层）

### C. Group（群组/编组）

#### C1. 创建/嵌套
- [ ] 多选 ≥2：可以编组
- [ ] 单选 group：可以“再包一层”生成第三层群组
- [ ] 单选非 group：不允许编组（符合产品规则）

#### C2. bounds 正确性
- [ ] 编组包含嵌套群组时，新群组 bounds 覆盖所有子孙
- [ ] 编组包含边时，新群组 bounds 覆盖边路径（waypoints）并留 padding
- [ ] 编组后不发生“跳到画布中间”

### D. 容器 resize 与边

- [ ] 拖动 group/frame 左上角 resize：内部节点/边的绝对位置不变
- [ ] resize 时边的折线不发生诡异变化
- [ ] 拖动容器移动：内部边 waypoints 随容器 delta 平移

### E. Edge（边）

- [ ] 单击边：弹出边菜单（延迟打开不误触双击）
- [ ] 双击边 label：只出现文字编辑工具条，其他菜单不出现/不复活
- [ ] 边菜单：
  - [ ] Label 开关可控制是否显示
  - [ ] 类型（smoothstep/bezier）可切换
  - [ ] 箭头类型（无/终点/起点/双向）可切换
  - [ ] 线条颜色/线宽可改
  - [ ] 动画开关可用

### F. 素材（Asset）

- [ ] 导入 PNG：显示正常、可缩放
- [ ] 导入 SVG：显示正常、可缩放
- [ ] SVG 颜色覆盖：使用 ColorEditor 拾色板，颜色变化正确
- [ ] 旋转：±45° 步进工作正常
- [ ] 翻转：水平/垂直翻转工作正常，可与旋转叠加

### G. 触摸板（Mac）

- [ ] 双指滚动平移画布稳定
- [ ] pinch 缩放稳定（不会偶发缩放失效/页面滚动抢占）

### H. 发布环境

- [ ] 生产环境箭头 marker 不丢（边带箭头时可见）
