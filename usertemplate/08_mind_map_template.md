# Mind Map Template（思维导图）

## 适用场景
适用于：主题分支展开、层级联想、脑图结构整理、把一段自然语言/文档整理成树状结构。

## 模板目标
生成一张思维导图：
- 只有节点（每一层都是节点）
- 层级用父->子边表达
- 整张图只需要 1 个外层画框（wrapper），不使用内部画框/嵌套分组

## 强约束
1. 只输出 1 个最外层 `subgraph` 作为画框（外框），subgraph 标题建议为：`思维导图`。
2. 除外框外：不允许任何嵌套 `subgraph`（每一层必须只用普通节点承载）。
3. 必须使用 `flowchart LR`（由系统强制校验）。
4. 必须形成树状层级：父节点通过边指向子节点，避免回边/环。
5. 节点 label 必须为中文且尽量短；语义太长时用 V2：`主标题｜副标题`。
6. 布局与样式由系统 `mind-map` 布局器负责：
   - 列式排版：根在最左列，深度越深越往右；同列纵向排列
   - 列间距目标约 8 个 grid units
   - 节点描边按“深度层级”顺序取主题色，描边粗细 strokeWidth=2
   - 边使用贝塞尔曲线效果

## 输出倾向
- 节点数建议 8~24 个
- 每层节点 1~5 个，避免过密失控
