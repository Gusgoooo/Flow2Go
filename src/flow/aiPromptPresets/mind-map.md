请把你在后续输入框中粘贴/输入的自然语言需求或文档内容，整理成一张“思维导图（Mind Map）”。

【任务目标】
- 提炼中心主题（根节点）
- 将关键概念拆成层级分支（多层子节点）

【强制生成规则】
1. 只要输出一张思维导图：每一层都是节点，不要用任何嵌套画框/子画框（只能有一个最外层画框）。
2. 用父->子边表达层级（父节点连到子节点），形成树状结构；不要回边或环。
3. 布局会由系统的 `mind-map` 布局器完成，你只需要提供正确的父子层级边关系：
   - 根在最左列
   - 深度越深越往右
   - 列间距约 8 个 grid units
4. 样式规则由系统自动应用：
   - 每一层节点按深度顺序取主题色，描边粗细 strokeWidth=2
   - 边使用贝塞尔曲线效果

【用户补充材料】
（你可以直接粘贴一段需求/文档；你不用写任何 DSL，直接输入自然语言即可。）
